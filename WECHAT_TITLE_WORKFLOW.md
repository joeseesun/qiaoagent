## 🎯 工作流概述

### 工作流信息
- **名称**: 微信爆款标题创作
- **ID**: `wechat_title_creator`
- **目标**: 基于文章内容，生成5-10个高质量、多样化的微信公众号爆款标题

### 核心特点
- ✅ **专业的三阶段流程**：内容分析 → 标题创作 → 优化评分
- ✅ **多样化标题风格**：涵盖8+种标题类型
- ✅ **科学评分体系**：5维度评分，满分10分
- ✅ **敏感内容检测**：自动识别可能触发审核的表达
- ✅ **元素解构分析**：深度剖析推荐标题的成功要素

## 🤖 三个专业 Agent

### 1. ContentAnalyzer（内容分析师）

**角色定位**: 内容分析专家，快速识别文章核心价值点

**主要任务**:
- 提取3-5个核心关键词
- 识别目标受众画像（年龄、职业、兴趣、痛点）
- 分析内容的情感基调（严肃/活泼/实用/紧急等）
- 找出最具传播价值的3个要点
- 识别可以利用的热点、名人或流行梗

**输出示例**:
```
核心关键词：AI、教育、个性化学习、效率提升、未来趋势
目标受众：25-40岁教育工作者、家长、教育创业者
情感基调：实用+前瞻+积极
传播要点：
1. AI如何解决传统教育痛点
2. 个性化学习的实际案例
3. 未来教育的发展趋势
可利用热点：ChatGPT、教育改革、双减政策
```

### 2. TitleCreator（爆款标题创作大师）

**角色定位**: 微信生态资深内容策略师，标题点击率专家

**创作策略**:

#### 信息传递型
- 新闻速递式：「重磅！教育部发布AI教学新规」
- 数字锦囊式：「提升学习效率的7个AI工具」
- 实用指南式：「如何用AI打造个性化学习方案」

#### 好奇心与情感型
- 悬念式：「这个AI功能，90%的老师都不知道...」
- 提问式：「AI会取代老师吗？答案出乎意料」
- 反常识式：「别再死记硬背了！AI告诉你正确方法」
- 热点借势式：「ChatGPT爆火后，教育行业的3大变化」

#### 营销心理型
- 恐惧诉求：「不懂AI的老师，正在被时代抛弃」
- 稀缺性：「仅限今天！AI教育工具免费开放」
- 场景化：「凌晨3点，这位老师用AI改变了一切」

#### 标点符号强化
- 感叹号：「震惊！AI让学渣逆袭成学霸」
- 问号：「AI教育真的靠谱吗？」
- 省略号：「用了AI之后，我发现...」
- 引号：「"AI老师"来了，真人老师怎么办？」
- 特殊符号：「【独家】AI教育的5个真相」

**输出示例**:
```
1. 震惊！这个AI工具让学习效率提升300% (悬念+数字+感叹号)
2. 别再死记硬背了！AI告诉你正确学习方法 (反常识+行动召唤)
3. 凌晨3点，这位老师用AI改变了一切... (场景化+悬念+省略号)
4. AI会取代老师吗？答案出乎意料 (提问+反转)
5. 【独家】ChatGPT爆火后，教育行业的3大变化 (热点+数字+特殊符号)
6. 不懂AI的老师，正在被时代抛弃 (恐惧诉求)
7. 提升学习效率的7个AI工具，第5个绝了 (数字锦囊+悬念)
8. "AI老师"来了，真人老师怎么办？ (引号+提问)
```

### 3. TitleOptimizer（标题优化与评分专家）

**角色定位**: 微信内容运营顶级专家，点击率预测准确率95%+

**评分标准**（满分10分）:
- **吸引力**（3分）：能否立即抓住眼球
- **相关性**（2分）：与内容的匹配度
- **情感触动**（2分）：能否引发情感共鸣
- **传播潜力**（2分）：用户转发意愿
- **合规性**（1分）：是否符合平台规范

**输出内容**:

1. **推荐标题列表**（带评分和⭐️标记）
2. **标题技巧与点击率分析**
3. **推荐标题元素解构分析**
4. **敏感内容提醒表格**
5. **使用建议和A/B测试建议**

**输出示例**:
```markdown
# 为您生成的微信公众号爆款标题

## 推荐标题

1. ⭐️ (9.2分) 震惊！这个AI工具让学习效率提升300%
2. ⭐️ (8.8分) 凌晨3点，这位老师用AI改变了一切...
3. ⭐️ (8.5分) 别再死记硬背了！AI告诉你正确学习方法
4. (8.2分) AI会取代老师吗？答案出乎意料
5. (7.9分) 【独家】ChatGPT爆火后，教育行业的3大变化

## 标题技巧与点击率分析

1. **标题1**: 悬念+数字+感叹号 - 数字"300%"制造强烈对比，"震惊"引发好奇
2. **标题2**: 场景化+悬念+省略号 - "凌晨3点"构建具体场景，省略号制造悬念
3. **标题3**: 反常识+行动召唤 - "别再...了"打破常规认知，引发思考

## 推荐标题元素解构分析

### ⭐️ 标题1: 震惊！这个AI工具让学习效率提升300%

- **引导词**："震惊！" - 强烈的情感冲击，立即抓住注意力
- **核心词**："AI工具"、"学习效率" - 直击目标受众关注点
- **修饰词**："这个" - 制造神秘感和独特性
- **情感词**："震惊" - 触发惊讶和好奇情绪
- **行动词**："提升" - 暗示积极改变和实际效果
- **数字元素**："300%" - 具体量化，增强可信度和冲击力
- **结构分析**：感叹号开头 + 指示词 + 核心价值 + 量化效果，节奏紧凑有力
- **使用场景**：适合推广工具类、方法类内容，强调效果显著

### ⭐️ 标题2: 凌晨3点，这位老师用AI改变了一切...

- **引导词**："凌晨3点" - 具体时间点，构建真实场景
- **核心词**："老师"、"AI"、"改变" - 人物+工具+结果
- **修饰词**："这位" - 增加真实感和代入感
- **情感词**："改变了一切" - 触发好奇和期待
- **行动词**："用"、"改变" - 展示行动和结果
- **标点符号**：省略号 - 制造悬念，引发点击欲望
- **结构分析**：时间场景 + 人物 + 工具 + 悬念结果，故事化叙事
- **使用场景**：适合案例故事类、人物访谈类内容，强调真实性

## 敏感内容提醒

| 标题序号 | 潜在敏感表达 | 修改建议 |
|----------|--------------|----------|
| 1 | "震惊" | 可替换为"惊喜"、"发现"等更温和的表达 |
| 5 | "独家" | 属于绝对化用词，建议删除或替换为"深度解析" |

## 使用建议

1. **优先推荐**: 使用⭐️标记的前3个标题，预估点击率最高
2. **A/B测试**: 建议同时测试标题1和标题2，对比不同风格的效果
3. **时间选择**: 标题2适合晚间推送，标题1适合早间推送
4. **受众匹配**: 标题3更适合教师群体，标题4更适合家长群体
5. **合规提醒**: 避免使用"震惊"、"独家"等可能触发审核的词汇
```

## 🚀 使用方法

### 1. 访问系统

打开浏览器访问：`http://localhost:3000`

### 2. 选择工作流

在工作流下拉菜单中选择：**微信爆款标题创作**

### 3. 输入内容

在"输入主题"框中输入您的文章内容或核心信息，例如：

```
文章主题：AI在教育领域的应用

核心内容：
- AI可以实现个性化学习，根据每个学生的学习进度和特点定制学习方案
- 通过AI辅助，教师可以节省50%的批改作业时间
- AI教育工具已在全国200+所学校试点，效果显著
- 但AI无法完全取代教师，人文关怀和情感教育仍需真人教师

目标受众：教育工作者、家长、教育创业者
```

或者简单输入：

```
AI教育工具如何提升学习效率
```

### 4. 点击生成

点击"开始生成"按钮，系统将自动：
1. 分析内容（ContentAnalyzer）
2. 创作标题（TitleCreator）
3. 评分优化（TitleOptimizer）

### 5. 查看结果

等待30-60秒，您将看到：
- ✅ 5-10个专业标题
- ✅ 每个标题的评分和排序
- ✅ ⭐️推荐标题的详细解构分析
- ✅ 敏感内容提醒
- ✅ 使用建议和A/B测试建议

## 💡 标题创作技巧总结

### 黄金法则
1. **前7字最关键** - 核心信息必须在前7个字呈现
2. **长度控制在7-18字** - 太短信息不足，太长影响阅读
3. **包含核心关键词** - 确保与内容高度相关
4. **触动目标受众痛点** - 引发共鸣和点击欲望
5. **避免标题党** - 标题必须真实反映内容

### 8大标题类型

| 类型 | 特点 | 示例 |
|------|------|------|
| 新闻速递式 | 快速传递信息 | 「重磅！教育部发布AI教学新规」 |
| 数字锦囊式 | 具体可量化 | 「提升学习效率的7个AI工具」 |
| 提问式 | 引发思考 | 「AI会取代老师吗？」 |
| 悬念式 | 制造好奇 | 「这个功能，90%的人都不知道...」 |
| 反常识式 | 打破认知 | 「别再死记硬背了！」 |
| 热点借势式 | 蹭热点流量 | 「ChatGPT爆火后的3大变化」 |
| 场景化式 | 构建画面 | 「凌晨3点，这位老师...」 |
| 情感共鸣式 | 触动情绪 | 「每个孩子都值得被看见」 |

### 标点符号的妙用

| 符号 | 作用 | 示例 |
|------|------|------|
| ！ | 增强情感 | 「震惊！效率提升300%」 |
| ？ | 引发思考 | 「AI真的靠谱吗？」 |
| ... | 制造悬念 | 「用了之后，我发现...」 |
| "" | 突出重点 | 「"AI老师"来了」 |
| 【】 | 区分强调 | 「【独家】行业真相」 |
| \| | 分隔信息 | 「AI教育\|未来已来」 |

## ⚠️ 注意事项

### 合规性要求

❌ **禁止使用的绝对化用词**:
- 最、首、独家、唯一、第一
- 100%、完全、绝对、永久
- 国家级、世界级、顶级

❌ **避免的敏感表达**:
- 过度夸张的承诺
- 涉及政治、宗教的词汇
- 低俗、诱导、虚假的表达

✅ **推荐的替代表达**:
- "独家" → "深度解析"、"全面解读"
- "最好" → "优质"、"推荐"
- "100%有效" → "效果显著"、"广受好评"

### 标题党风险

❌ **标题党特征**:
- 标题与内容严重不符
- 夸大其词、歪曲事实
- 纯粹为了吸引点击

✅ **正确做法**:
- 标题真实反映内容核心
- 适度吸引但不夸张
- 提供实际价值

## 🧪 测试建议

### A/B测试方案

1. **风格对比测试**
   - 测试组A：悬念式标题
   - 测试组B：数字锦囊式标题
   - 对比点击率和转发率

2. **情感基调测试**
   - 测试组A：积极正面标题
   - 测试组B：恐惧诉求标题
   - 对比用户反馈

3. **长度测试**
   - 测试组A：7-10字短标题
   - 测试组B：15-18字长标题
   - 对比阅读完成率

### 效果评估指标

- **点击率（CTR）**: 标题吸引力的直接体现
- **阅读完成率**: 标题与内容的匹配度
- **转发率**: 内容价值和传播潜力
- **评论互动**: 用户参与度和共鸣度
